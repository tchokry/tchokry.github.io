Dance Notes

Home / Dance Notes



  • Goal: Create a functional web application that can enhance user experience through sound feedback
  • Date: January - March 2019

  • Skills: Needfinding, Prototypes, Wireframing, Programming, User Testing

  • Tools: HTML, CSS, Adobe XD, Heroku, Balsamic

  • Roles: UX & UI Designer

Project Overview

The goal of this project is to create a functional application that can enhance user experience through sound feedback. Dance Notes is the application that allow users to add audio cues on top of the music they are dancing to. This features will help dancers ingrain accurate counts to the beat into their mind without the help of an instructor as it guide them through silent portions of a musical choreography, or provide audio reminders to what a dancer needs to keep in mind while performing the piece.

Point of View

Learning how to dance is hard as it requires memorizing movement sets that are in time with the music.

Storyboards

Two different design ideas that address and engage the point of view to address the user needs.



Paper Parallel Prototype

After gaining some insights from our storyboards and our target users along with their problems, we made paper prototype that connect to our point of view that illustrate the significant & interactive elements that we hope to achieve in our application.

❮ ❯




Wireframes

Digital non-functional mockup of the mobile main content screen and highest priority screen of the web application. The main goal for this stage is to visualize all potential interaction via content, buttons, text boxes, or even functions for both of the home and play page screen




User Testing

To gain user feedbacks, we execute both in person and online A/B user testing in which the participants that resembles our target users were asked to do some tasks that reflects the core functionality of our application. The suggestions and feedback from these testings then taken into account to improve our web application.


Additionally, we use Google Analytics for A/B testing to determine which design of the home page that gives users better browsing experience.
